Description
An AI-powered platform that analyzes resumes against job descriptions, 
provides ATS scores, identifies skill gaps, and generates personalized learning roadmaps.

Features
- ATS Score Analysis
- Skill Gap Detection
- AI Resume Coach
- 3-Week Learning Roadmap
- Skill Simulation (what-if analysis)
- Mock Interview System
- History Tracking


 Tech Stack
- Backend: Flask (Python)
- Frontend: HTML, CSS, JavaScript
- AI: Gemini API
- NLP: Sentence Transformers
- Database: SQLite
  
 How to Run

1. Clone the repository:
   git clone https://github.com/your-username/your-repo-name.git

2. Install dependencies:
   pip install -r requirements.txt

3. Run the server:
   python app.py

4. Open in browser:
   http://127.0.0.1:5000
